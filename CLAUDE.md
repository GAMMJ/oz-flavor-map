# 맛집 프로젝트

## 프로젝트 개요
위치 기반 맛집 탐색 서비스. 현재 위치에서 가까운 맛집을 조회하고, 마음에 드는 곳을 찜할 수 있다.

## 기술 스택
- **프레임워크**: React 19
- **언어**: JavaScript
- **빌드 도구**: Vite
- **배포**: AWS

## 주요 기능
- 전체 맛집 정보 조회
- 현재 위치 기반 거리순 정렬
- 맛집 찜하기 / 찜 목록 조회 / 찜 삭제

## 미션 단계
| 단계 | 내용 |
|------|------|
| 1단계 | 전체 맛집 목록 조회 |
| 2단계 | 현재 위치 기반 거리순 정렬 |
| 3단계 | 찜하기 / 찜 목록 조회 / 찜 삭제 |
| 4단계 | AWS 배포 |

## 커밋 컨벤션
기능 단위로 커밋을 나눈다. 한글을 기본으로 하되, 영어가 더 자연스러운 경우 영어를 사용한다.

### 타입
| 타입 | 설명 |
|------|------|
| `feat` | 새로운 기능 추가 |
| `fix` | 버그 수정 |
| `style` | 스타일 변경 (기능 무관) |
| `refactor` | 리팩토링 |
| `chore` | 설정, 패키지, 기타 |
| `docs` | 문서 수정 |

### 예시
```
feat: 전체 맛집 목록 조회 기능 추가
feat: 현재 위치 기반 거리순 정렬 구현
feat: 찜하기 기능 추가
feat: 찜 목록 조회 기능 추가
feat: 찜 삭제 기능 추가
chore: AWS 배포 설정
```

## PR 컨벤션
- 제목: `n단계 미션 해결`
- 해당 단계의 커밋들을 묶어서 PR 생성
- 예시: `1단계 미션 해결`, `2단계 미션 해결`
